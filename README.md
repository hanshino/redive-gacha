# redive-gacha
Princess Connect Re:dive Gacha Simulator


### Requirements

1. Node.js

### How to use it?

1. open your command
2. `git clone https://github.com/hanshino/redive-gacha.git`
3. `cd redive-gacha`
4. `npm install` or `yarn install`
5. `node index.js`
